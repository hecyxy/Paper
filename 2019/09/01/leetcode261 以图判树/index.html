leetcode261 以图判树

发表于 2019-09-01 | 分类于 算法与数据结构 | 阅读数

给n个节点[0,n-1]，和一个边的数组，检查其是否是一个树结构。

For example:

Given n = 5 and edges = [[0, 1], [0, 2], [0, 3], [1, 4]], return true.

Given n = 5 and edges = [[0, 1], [1, 2], [2, 3], [1, 3], [1, 4]], return false.

给出了无向图，判断是否为一棵树，如果是树，所有节点都必须是连接的，而且不能有环。工作就变成了：验证是否是连通图、是否有环。
